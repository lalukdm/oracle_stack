7. SQL Скрипт создания БД (аналог примера)

CREATE DATABASE XE
MAXINSTANCES 1
MAXLOGHISTORY 1
MAXLOGFILES 5
MAXLOGMEMBERS 3
MAXDATAFILES 100
DATAFILE '/u01/app/oracle/oradata/XE/system.dbf' SIZE 360M REUSE AUTOEXTEND ON NEXT 10M MAXSIZE UNLIMITED
EXTENT MANAGEMENT LOCAL
DEFAULT TEMPORARY TABLESPACE TEMP
TEMPFILE '/u01/app/oracle/oradata/XE/temp.dbf' SIZE 20M REUSE AUTOEXTEND ON NEXT 5M MAXSIZE UNLIMITED
UNDO TABLESPACE "UNDOTBS1" DATAFILE '/u01/app/oracle/oradata/XE/undotbs1.dbf' SIZE 25M REUSE AUTOEXTEND ON NEXT 5M MAXSIZE UNLIMITED
CHARACTER SET AL32UTF8
NATIONAL CHARACTER SET UTF8
LOGFILE GROUP 1 ('/u01/app/oracle/oradata/XE/redo01.log') SIZE 50M,
        GROUP 2 ('/u01/app/oracle/oradata/XE/redo02.log') SIZE 50M,
        GROUP 3 ('/u01/app/oracle/oradata/XE/redo03.log') SIZE 50M;
Примечание: Отчет создан на основе реальной конфигурации Oracle XE. Параметры могут отличаться от примера satskov, но структура отчета соответствует требованиям.
