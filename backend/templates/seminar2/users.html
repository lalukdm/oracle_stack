Дата выполнения: 2026-03-07

Задание: Управление профилями пользователей с правами DBA. Создание пользователей, назначение привилегий и ролей.

1. Создание пользователей (CREATE USER)

Пример 1: Простой пользователь

CREATE USER asu_user1 IDENTIFIED BY asu123 DEFAULT TABLESPACE USERS TEMPORARY TABLESPACE TEMP QUOTA 10M ON USERS ACCOUNT UNLOCK;

Пример 2: Пользователь с неограниченными квотами (как в методичке)

CREATE USER asu_test IDENTIFIED BY test123 PROFILE DEFAULT DEFAULT TABLESPACE USERS QUOTA UNLIMITED ON USERS QUOTA UNLIMITED ON SYSTEM QUOTA UNLIMITED ON TEMP ACCOUNT UNLOCK;

Пример 3: Административный пользователь

CREATE USER asu_admin IDENTIFIED BY admin123 DEFAULT TABLESPACE USERS TEMPORARY TABLESPACE TEMP QUOTA UNLIMITED ON USERS PROFILE DEFAULT ACCOUNT UNLOCK;

2. Назначение привилегий (GRANT)

2.1. Системные привилегии

GRANT CREATE SESSION TO asu_user1; GRANT CREATE TABLE TO asu_user1; GRANT CREATE VIEW TO asu_user1; GRANT CREATE SEQUENCE TO asu_user1; GRANT CREATE PROCEDURE TO asu_user1; GRANT CREATE TRIGGER TO asu_user1;

2.2. Назначение ролей

GRANT CONNECT TO asu_test; GRANT RESOURCE TO asu_test; GRANT DBA TO asu_admin;

3.1. Системные привилегии

SELECT GRANTEE, PRIVILEGE, ADMIN_OPTION FROM DBA_SYS_PRIVS WHERE GRANTEE IN ('ASU_USER1', 'ASU_TEST', 'ASU_ADMIN') ORDER BY GRANTEE, PRIVILEGE;

4. Создание и управление ролями

4.1. Создание пользовательских ролей

CREATE ROLE asu_developer_role NOT IDENTIFIED; CREATE ROLE asu_analyst_role IDENTIFIED BY analyst123; CREATE ROLE asu_manager_role;

4.2. Назначение привилегий ролям

-- Для разработчиков GRANT CREATE SESSION, CREATE TABLE, CREATE VIEW, CREATE SEQUENCE, CREATE PROCEDURE TO asu_developer_role; GRANT SELECT ANY TABLE TO asu_developer_role; -- Для аналитиков GRANT CREATE SESSION TO asu_analyst_role; GRANT SELECT ANY TABLE TO asu_analyst_role; -- Для менеджеров GRANT CREATE SESSION TO asu_manager_role; GRANT SELECT ANY TABLE, INSERT ANY TABLE, UPDATE ANY TABLE, DELETE ANY TABLE TO asu_manager_role;

4.3. Назначение ролей пользователям

GRANT asu_developer_role TO asu_user1; GRANT asu_analyst_role TO asu_test; GRANT asu_manager_role TO asu_admin;

5. Изменение пользователей (ALTER USER)

5.1. Смена пароля

ALTER USER asu_user1 IDENTIFIED BY newpass123;

5.2. Блокировка/разблокировка

ALTER USER asu_test ACCOUNT LOCK; ALTER USER asu_test ACCOUNT UNLOCK;

5.3. Изменение квот

ALTER USER asu_user1 QUOTA 20M ON USERS;

6. Информация о пользователях

SELECT USERNAME, ACCOUNT_STATUS, DEFAULT_TABLESPACE, TEMPORARY_TABLESPACE, CREATED FROM DBA_USERS WHERE USERNAME IN ('ASU_USER1', 'ASU_TEST', 'ASU_ADMIN') ORDER BY USERNAME;
