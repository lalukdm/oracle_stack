SQL запрос с ROLLUP:

SELECT 
    CASE 
        WHEN GROUPING(e.MANAGER_ID) = 1 THEN NULL 
        ELSE e.MANAGER_ID 
    END AS manager_id,
    CASE 
        WHEN GROUPING(e.JOB_ID) = 1 THEN NULL 
        ELSE e.JOB_ID 
    END AS job_id,
    COUNT(*) AS employee_count,
    SUM(e.SALARY + NVL(e.COMMISSION_PCT, 0) * e.SALARY) AS total_payment
FROM HR.EMPLOYEES e
WHERE e.MANAGER_ID IS NOT NULL
GROUP BY ROLLUP (e.MANAGER_ID, e.JOB_ID)
ORDER BY e.MANAGER_ID, e.JOB_ID;

Объяснение:

  • GROUP BY ROLLUP (manager_id, job_id) - создает три уровня итогов:
    • Детальные строки (manager_id + job_id)
    • Подытоги по каждому manager_id (job_id = NULL)
    • Общий итог (manager_id = NULL, job_id = NULL)
  • GROUPING() - определяет, является ли поле итоговым (1) или детальным (0)
  • SUM(e.SALARY + NVL(e.COMMISSION_PCT, 0) * e.SALARY) - оклад + комиссионные
  • NVL(commission_pct, 0) - заменяет NULL на 0
