📅 Задание 2.6: Номер дня недели

Вычисление номера дня недели (понедельник = 1) независимо от NLS-установок

{% if error %}
{{ error }}
{% endif %} {% if result %}

Результат:

Текущая дата: {{ result.current_date }}
День недели (англ.): {{ result.day_name }}
Номер дня (пн=1): {{ result.day_number }}
{% endif %}

Каноническое решение:

SELECT (INSTR('MONTUEWEDTHUFRISATSUN', 
              TO_CHAR(SYSDATE, 'DY', 'NLS_DATE_LANGUAGE = AMERICAN')) + 2) / 3 
FROM DUAL;

Как это работает:

  1. TO_CHAR(SYSDATE, 'DY', 'NLS_DATE_LANGUAGE = AMERICAN') - получаем сокращенное название дня недели на английском (MON, TUE, WED...)
  2. INSTR('MONTUEWEDTHUFRISATSUN', ...) - ищем позицию этого названия в строке
  3. (позиция + 2) / 3 - преобразуем позицию в номер дня (MON=1, TUE=2...)

Проверка для всех дней:

День Позиция Формула Номер
MON1(1+2)/31
TUE4(4+2)/32
WED7(7+2)/33
THU10(10+2)/34
FRI13(13+2)/35
SAT16(16+2)/36
SUN19(19+2)/37
